# AesCipherFeedback256

A simple yet powerful code for encrypting files or data.
We didn't use any imports or includes.
Pure C++ and Java.
It was based on [(Cipher_Aes.java)](https://github.com/openjdk/jdk/blob/303de9a3f2ba93f0bbe42044483a0b48c82b70cb/src/java.base/share/classes/com/sun/crypto/provider/AES_Crypt.java) a Official Java file from internal JDK.

Um código simples, mas poderoso para criptografar arquivos ou dados.
Não usamos nenhuma importação ou inclusão.
C++ e Java puro.
Foi baseado em [(Cipher_Aes.java)](https://github.com/openjdk/jdk/blob/303de9a3f2ba93f0bbe42044483a0b48c82b70cb/src/java.base/share/classes/com/sun/crypto/provider/AES_Crypt.java) um arquivo oficial do JDK interno.

Testados no CLion e no InteliJ IDEA.
Testados em arquivo de texto e em video mp4. Ambos emcriptados e decriptados com sucesso.

O código funciona também no JNI do android, como linguagem nativa(lib.so).
